How to play the game:
1. Run the Shmup.exe file
2. In the game, use WASD or arrow keys to move.
3. Use CTRL or LEFT CLICK to shoot.
4. Press ESC to exit the program.

~Source codes~
DamageDealer
- Represents an entity that causes damage. 

Enemy
- Handles enemy attack and damage.

EnemyPathing
- Handles the movement of enemies towards waypoints.

EnemySpawner
- Spawns enemies for all waves and per each wave.

MusicPlayer
- Plays a single music track.

Player
- Handles player movement, attack, and damage.

Shredder
- Object that destroys any projectiles that go off-screen.

WaveConfig
- Scriptable object that represents each wave of enemies.
